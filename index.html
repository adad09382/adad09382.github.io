概念

E-Firend是一個基於AI的英語口說對話平台，希望可以為預算有限，又想要練習英語 口說的人們提供一個選擇，使用者可以在"探索話題"模塊找到不同的對話話題， 點下LET'S TALK開始與AI就選擇的話題開始對話交流。此外本網站還提供了"新聞話題"區塊， 可以在之中找到感興趣的新聞，並與AI進行討論互動!

前台功能
  • 會員登入註冊
  • 陳列 News API 新聞返回的內容
  • 選擇話題與AI開始對話
  • 首次進入話題與AI收到信息時會發送消息，點擊AI語音撥放鍵開始語音合成
  • 在對話窗口點擊麥克風按鈕開始語音識別與錄音
  • 記錄對話歷史紀錄，可重複撥放歷史錄音與AI的回覆
後台管理系統
  • 管理員帳號可進入後台
  • 可增改刪查對話Prompt
